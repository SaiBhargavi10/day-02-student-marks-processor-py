# Student Marks Processor

## Overview
A Python console application that reads student marks from a CSV file,
calculates total marks, percentage, and assigns grades based on predefined rules.

This project focuses on clean coding practices, file handling,
and basic data validation.

## Features
- Reads structured student data from CSV
- Calculates total marks and percentage
- Assigns grades based on clear rules
- Validates marks to avoid incorrect data
- Displays a clean, readable report

## Grading Criteria
- Percentage ≥ 90 → Grade A
- Percentage ≥ 75 → Grade B
- Percentage ≥ 50 → Grade C
- Percentage < 50 → Grade F

## Tech Stack
- Python 3
- CSV File Handling
